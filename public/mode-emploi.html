Guide Officiel

Mode d'emploi complet

Toutes les astuces pour utiliser le site sans erreur: matchs, details, coupon, image, PDF, Telegram et gestion des problemes.

1. Demarrage rapide

Etape A

Ouvre la page d'accueil, choisis une ligue, puis un mode: A venir, Turbo Top 10, En cours.

Etape B

Clique Detail predictions pour voir l'analyse complete du match (moteur IA + marches + validation).

Etape C

Va sur Page Coupon, regle le nombre de matchs, la ligue et le profil de risque, puis genere.

2. Comprendre les modes

A venir Matchs pre-match, ideal pour coupons propres.

Turbo Top 10 Les meilleurs pre-matchs classes par fiabilite.

En cours Matchs deja lances (utiles pour lecture, pas ideal pour coupon propre).

Termines Historique des matchs.

3. Page detail match

  • Neural Match Engine: radar + flux dynamique + KPI.
  • Validation locale: aide a voir si le pick reste propre.
  • Envoyer Telegram: envoi rapide d'un ticket 1 match.
  • PDF Match: export du ticket detail match.

4. Page coupon (toutes les options)

Generation

  • Generer Coupon: 1 coupon selon ton profil.
  • 3 Strategies: Safe / Equilibre / Agressif.
  • Remplacer Pick Faible: swap intelligent du pick le moins fort.

Controle qualite

  • Valider Ticket Pro: verification drift + marche + statut match.
  • Ticket Shield: bloque les selections dangereuses.
  • Alerte correlation: signale si le coupon est trop lie.

Exports

  • PDF Rapide: version courte.
  • PDF Resume: version standard.
  • PDF Detaille: version analytique + plan B.
  • Image Coupon: carte visuelle lisible style accueil.

5. Telegram (texte, image, pack)

  • Envoyer Telegram: envoi texte.
  • Telegram Image: envoi image coupon lisible.
  • Envoi Groupe 1-Clic: envoie texte + image + PDF en une action.

Si aucun chat n'est detecte, ecris d'abord un message au bot Telegram puis reessaie.

6. Parametres recommandes (propre)

Profil: Equilibre Nombre de matchs: 2 a 4 Seuil drift: 5% a 7% Mode matchs: A venir ou Turbo Top 10 Toujours: Valider Ticket Pro avant envoi/export

7. Resolution des erreurs frequentes

  • Route API introuvable: redemarre le serveur puis recharge la page en dur (Ctrl+F5).
  • Erreur JSON/HTML: ancien frontend cache, fais un hard refresh.
  • Match deja demarre: retire le pick ou regenere en mode A venir.
  • Telegram echec: verifie TELEGRAM_BOT_TOKEN et qu'un chat existe avec le bot.
  • Bouton grise: genere d'abord un coupon non vide.

8. Bonnes pratiques de discipline

  • Ne jamais forcer un coupon si la validation demande correction.
  • Eviter les coupons trop longs; preferer qualite a quantite.
  • Surveiller les drifts forts avant envoi final.
  • Aucune combinaison n'est garantie gagnante.

9. FAQ rapide

Q: Pourquoi je ne vois pas certains matchs ?
R: Filtre ligue/mode actif, ou match pas dans le flux courant.

Q: Pourquoi un pick est remplace ?
R: Ticket Shield detecte un risque (drift, marche indisponible, faible confiance, match demarre).

Q: Je veux partager vite ?
R: Utilise Envoi Groupe 1-Clic.
